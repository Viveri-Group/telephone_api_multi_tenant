# Telephony API

A Laravel-based backend API for managing competitions, phone lines and telephone calls. The project includes a built-in admin panel for monitoring and managing:

1. Call activity (live and historical).
2. Phone line configurations.
3. Competitions and related metadata.

### Features

- RESTful API for core telephony operations
- Admin interface for non-technical oversight and data management
- This repository serves as a backend foundation for systems involving inbound/outbound call flows and related operational workflows.
